# Escritor Experto Evolutivo - Recursos y Referencias

Este directorio contiene recursos complementarios para la skill de escritura evolutiva.

## Archivos

- **SKILL.md** - Skill principal con técnicas, modes y workflows
- **EVOLUTION_LOG.md** - Registro de aprendizajes y evolución continua
- **README.md** - Este archivo

## Uso

### Activar la Skill

La skill se activa automáticamente cuando:
- Usuario menciona: "escribir", "mejorar texto", "redactar", "contenido"
- Se trabaja con archivos de texto (.md, .txt, .docx)
- Se solicita narrativa, historias o copywriting

### Actualizar la Skill

Después de cada proyecto de escritura:

1. Añadir entrada en `EVOLUTION_LOG.md`
2. Actualizar técnicas en `SKILL.md` si es necesario
3. Commit cambios con mensaje descriptivo

### Integración con Proyecto

Esta skill fue creada y refinada durante el proyecto "Reflexión Social", donde se transformaron 10 capítulos de estilo académico a contenido viral para Wattpad.

**Resultados:**
- 10 capítulos mejorados
- Expansión del 30-50%
- Hooks impactantes implementados
- Estructura optimizada para mobile
- Tono coherente en todo el libro

## Principios Core

1. **Evolución Continua** - Cada proyecto mejora la skill
2. **Adaptabilidad** - Ajuste según medio y audiencia
3. **Engagement Primero** - Sin atención, no hay mensaje
4. **Edición Despiadada** - Menos es más
5. **Especificidad** - Lo genérico es olvidable

## Modes Disponibles

1. **Narrativa Viral** - Wattpad, Medium, Social Media
2. **Profesional** - Artículos, Ensayos, Papers
3. **Ficción** - Cuentos, Novelas, Narrativa
4. **Copywriting** - Sales, Marketing, Persuasión

## Contacto y Mejoras

Esta skill está diseñada para ser mejorada continuamente. Si descubres nuevas técnicas, añádelas al Evolution Log y actualiza la skill principal.
