# Workflow: Captura Visual y Portafolio

Convierte el código en material visual "Premium".

1. **Preparación**: Asegurar que la aplicación esté corriendo (usualmente `npm run dev` o `php artisan serve`).
2. **Entorno**: Verificar que `capture-script.js` esté presente en la raíz.
3. **Ejecución**:
   // turbo
   `node capture-script.js`
4. **Revisión**: Analizar las imágenes en la carpeta `captures/`.
5. **Post-Proceso**: Si es necesario, usar la herramienta de edición para añadir frames o sombras (ver `experto-capturas-visuales`).
6. **Entrega**: Presentar al usuario un resumen visual de las capturas realizadas.
